D) Changing the URL Path of the Data - top



In Project Explorer, Locate Config.java file under RestaurantLocator > src >com.config package then select and open Config.java



Change the approriate URL Path of the XML/JSON/PHP File file you uploaded in the server.



E) Changing Data File Format(JSON/XML) - top



In the same file, Config.java, locate WILL_USE_JSON_FORMAT

Changing this to YES will use JSON Data Format

Changing this to NO will use XML Data Format



F) Adding Entry in JSON File Format (Restaurants) (Optional) - top



In Restaurant Locator folder, Go to Server File folder and open the JSON file using your favorite text editor

In the restaurants object after the [ and before the ] place the following, separated by comma each entry

Fill up with your data inside the blank quotes

"restaurant_id", "featured", "created_at", "category_id" requires INTEGER VALUES inside e.g. "3"

"food_rating", "price_rating" requires FLOAT VALUES inside e.g. "3.5"

"lat", "lon" requires DOUBLE VALUES inside e.g. "125.3441"

and the rest requires TEXT VALUES inside e.g. "Good to see."
