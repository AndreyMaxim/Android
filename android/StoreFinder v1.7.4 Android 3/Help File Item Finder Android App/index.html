F) Changing Twitter Consumer Key/Secret - top



In Project Navigator Pane, Locate Config.java under com.config package



Change the approriate Consumer Key/Secret that creating a twitter application provided.



G) Changing Facebook ID - top



In Project Navigator Pane, Locate strings.xml under res/values/



Change the approriate Facebook ID upon facebook provided when you create apps via Facebook Developer Portal



H) AdMob Show/Hide - top



In Project Navigator Pane, Locate Config.java under com.config package



Change WILL_SHOW_ADS to true to display ads, false if not.
