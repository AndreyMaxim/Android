A) Installing MAMP/WAMP - top

B) Creating a Database - top



Create a database and named as "db_storefinder" and select collation as "utf8_general_ci"



C) Importing the SQL File - top



The file SQL File found at the Store Finder Android v1.7/MySQL SQL File/




Click the db_storefinder then select Import




Select the db_storefinder.sql then click Open




Press the Go Button then wait for confirmation




IF all is working good then you will see all the tables in the Structure Tab



D) Changing the Config File - top



You can locate the config file at this folder Store Finder Android v1.7/PHP Scripts/storefinder/application/config.php



Open the file in your favorite editor then change the content of it



Modify the content inside the red boxes depending on your settings of your server or your localhost

The ROOT_URL below should be change depending on how the server access your root directory.



E) Changing the Global File - top



You can locate the global file at this folder Store Finder Android v1.7/PHP Scripts/storefinder/application/Global.php



Open the file in your favorite editor then change the content of it



Modify the content inside the red boxes depending on your needs

The DEBUG can be set to true if you wish to display any errors or warnings

The KEY_SALT can be set to any alphanumeric values to make your server more secure.
